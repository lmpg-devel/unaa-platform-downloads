# UNAA Platform Downloads

Depot public des artefacts telechargeables de la plateforme UNAA.

Ce depot ne contient pas les sources privees. Il ne doit contenir aucun secret, token,
mot de passe, keystore ou chemin local.

## Contenu

- `apps/` : pages publiques de lancement ou de telechargement.
- `data/` : donnees publiques exportees par les modules.
- `releases/` : emplacement reserve aux artefacts versionnes si la publication GitHub Release est miroir.
- `unaa-suite.latest.json` : manifeste public de la suite.
